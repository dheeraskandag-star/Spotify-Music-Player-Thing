# Spotify-Music-Player-Thingy

It is a music palyer which uses spotify's api to control song playback. 

It has three butons to play/pause,next,and previous.With a display to show current song.

I made it because i had to open spotify everytime to change songs. With this,I can change songs easily and it rests on my desk.
